Paragrafos e quebras de linhas

Um paragrafo pode ser escrito colocando no meio do par de tags (<P>, </P>)

Um paragrafo pode ser quebrado
usando a tag (<BR>) m

Voce pode adicionar varios simbolos especiais como:
® = (&R E G)
© = (&C O P Y)
™ = (&TRADE)
€ = (&EURO)
£ = (&POUND)
¢ = (&CENT)

Voce pode tambem adicionar emojis como:
🤔 = (&#x 1F914)
😏 = (&#x 1F60F)
😎 = (&#x 1F60E)
